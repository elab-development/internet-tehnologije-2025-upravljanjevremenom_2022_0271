# HandleTime - Platforma za upravljanje vremenom

## Opis aplikacije
Aplikacija omogućava korisnicima da efikasno organizuju svoje obaveze putem zadataka i podsetnika. Sistem razlikuje tipove korisnika (Student, Premium, Običan korisnik) i pruža personalizovan interfejs u skladu sa ulogom.

## Tehnologije
- **Backend:** Laravel (PHP)
- **Frontend:** React.js (Vite)
- **Baza podataka:** MySQL
- **Stilizacija:** Tailwind CSS
- **Autentifikacija:** Laravel Sanctum
